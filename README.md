## Campus Voting Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6a9a192a5eb04a9c8404dcaffe26b61d/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, campus administrators, system administrators.

**Regular User Features:**
1. Login
2. Personal Center
3. Unit Management
4. Jump Push
5. Voting Information
6. Notification Reply

**Campus Administrator Features:**
1. Login
2. Personal Center
3. User Management
4. Unit Category Management
5. Unit Management
6. Notification Push Management
7. Voting Information Management
8. Notification Reply

**System Administrator Features:**
1. Login
2. Personal Center
3. Institution Management
4. User Management
5. Unit Category Management
6. Institution Administrator Management
7. Unit Management
8. Notification Push Management
9. Voting Information Management
10. Notification Reply Management
